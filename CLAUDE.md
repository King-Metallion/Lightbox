## Project Overview

This is the documentation/demo site for [Lightbox2](https://github.com/lokesh/lightbox2), a JavaScript image overlay library. The site lives at lokeshdhakar.com/projects/lightbox2/. It is a static site (single `index.html` page) — there is no build pipeline, bundler, or framework.

## Commands

- `npm run dev` — Start local dev server (http-server)
- `npm run build` — Copy lightbox2 dist files from node_modules into `vendor/lightbox2/`
- `npm run deploy` — SFTP deploy to Dreamhost (requires `.host.json` config, see `.host.json.example`)

## Architecture

- **`index.html`** — The entire site: demos, usage docs, options table, API reference, theming guide. Uses Prism.js (loaded from CDN) for code highlighting.
- **`css/screen.css`** — Site styles. `css/prism.css` is the syntax highlighter theme.
- **`js/github.js`** — ES module that fetches the GitHub star count for the repo and displays it in the header.
- **`vendor/lightbox2/`** — Local copy of lightbox2 dist assets (CSS, JS, images). Generated by `npm run build` from the `lightbox2` npm package. This directory is committed to the repo.
- **`build/deploy.js`** — Node script that SFTPs the site to Dreamhost. Reads credentials from `.host.json`. Excludes `.git`, `node_modules`, `build`, and config files.

## Key Details

- jQuery is bundled via `lightbox-plus-jquery.min.js` — no separate jQuery dependency needed.
- The inline `<script>` at the bottom of `index.html` uses jQuery (`$`) for smooth-scrolling nav behavior.
- Google Analytics (gtag.js) is included inline in `index.html`.
